# BlackERP Community

ERP SaaS multiempresa desenvolvido para pequenas empresas.

## Funcionalidades

- Autenticação JWT
- Gestão de Empresas
- Gestão de Usuários
- Controle de Estoque
- Cadastro de Produtos
- Cadastro de Clientes
- Registro de Vendas
- Dashboard

## Tecnologias

### Backend
- Java
- Spring Boot
- Spring Security
- PostgreSQL
- Flyway
- Docker

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS

## Arquitetura

- Controller
- Service
- Repository
- DTO
- Entity
